# toronto-coffee-sales-analysis

## Project Overview
This project analyzes sales data for coffee shop chain in Toronto to uncover revenue trends, customer purchasing behavior, and product performance using SQL.

## Business Objectives
- Measure total sales revenue
- Analyze customer order frequency
- Identify top-selling products
- Identify high-value customers

## Dataset Description
The dataset includes customer records and sales orders with information on products sold, order dates, and transaction amounts.

## Tools Used
- SQL (PostgreSQL)
- pgAdmin

## Analysis Highlights
- Aggregated total revenue and customer spending
- Ranked products by sales performance
- Identified repeat and high-value customers

## Project Structure
- 'sql/' - SQL queries used for analysis
- 'screenshots/' - Query outputs and report visuals
